User Guide

Everything you need to know to get the most out of Azu.

Getting Started

1. Download & install

Download the .exe installer from Releases. Run it. Azu installs to your user profile — no admin needed.

2. Your first split

Press Ctrl+Shift+H to split horizontally or Ctrl+Shift+V vertically. Each pane gets its own shell.

3. Set project directories

Click the folder icon in each pane's toolbar. Select a project directory. The pane's shell will cd there automatically.

4. Save your layout

Click "Layouts" → type a name → "Save Layout". Your pane arrangement, directories, and labels are preserved.

5. Launch AI agents

Click the ▶ button on any pane → choose Claude, Codex, etc. Or use "▶ All" to launch in every pane simultaneously.

Pane Management

Split

Toolbar buttons or Ctrl+Shift+H/V. The original terminal session is preserved.

Close pane

Click ✕ or press Ctrl+W.

Swap positions

Click the ↕ icon → select which pane to swap with.

Rename

Double-click the pane label to edit. Or let Azu auto-rename from the running process.

Per-pane theme

Click the half-circle icon (◐) → select a theme just for that pane.

Features

Opacity

Drag the slider in the title bar. Only backgrounds become transparent — text stays readable.

Snippets

Click "⚡ Snippets" → "+ Add snippet" → name + command. Click to run in all shells.

Environment

Click "⚡ Env" in the status bar. Shows installed tools with versions.

Image paste

Copy a screenshot, press Ctrl+V in terminal. The image is saved as a temp file and the path is typed automatically.

State persistence

Azu auto-saves your layout, theme, and pane setup every 30 seconds. On reopen, everything is restored.
